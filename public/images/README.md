# Images Required for BOLD! Parallax Component

Place the following images in this directory:

## Required Images:

### Logo
- `logo.png` - The BOLD! CREATIVE STUDIO logo (should be a PNG with transparent background)

### Portfolio Images
Create a `portfolio` subfolder and place these images:

- `portfolio/bokija.jpg` - Bokija project showcase
- `portfolio/ayesha-kanwal.jpg` - Dr. Ayesha Kanwal project showcase
- `portfolio/odorguard.jpg` - odorguard project showcase
- `portfolio/neuvian-skincare.jpg` - Neuvian Skincare project showcase
- `portfolio/shireen-khoana.jpg` - Shireen Khoana project showcase

## Image Specifications:

- **Logo**: Recommended size: 400x200px or larger, PNG format with transparency
- **Portfolio Images**: Recommended size: 800x800px (square), JPG or PNG format

## Folder Structure:
```
public/
  images/
    logo.png
    portfolio/
      bokija.jpg
      ayesha-kanwal.jpg
      odorguard.jpg
      neuvian-skincare.jpg
      shireen-khoana.jpg
```

Note: The component includes fallback styling for missing images, so it will still display even if some images are not available.
